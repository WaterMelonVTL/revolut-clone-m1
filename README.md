# Entrepot de donnée pour un clone de Revolut
GHOBARA,VERAN,BERGON projet M1 EDBD

## Objectifs à maximiser : 

### Analyse des transactions

Objectif : Lutter contre la fraude
Fait analysé : Transactions avec flag frauduleuse, montant etc...
Dimensions : Client, Temps, Lieu, Type de transaction, catégrie de transaction, plan, canal, devise 

### Analyse des investissements dans les produits financiers

Objectif : Negocier des partenariats avec certains emmeteurs
Fait analysés : Achats de produits financiers (ETF, Actions, obligations, matières premières
Dimentions : Frais, Emmeteurs, Type de produit, Client, Temps, Lieu, plan, canal, devise 

### Analyse des crédits/prets

Objectif : Adapter les frais et choisir à quel profil de client on donne des prets. 
Fait analysé : Crédits passés
Dimentions : Client, Lieu, Temps, produit de crédit, plan, canal, devise
